# PHR Vault (Flask)# PHR Vault (Flask)

PHR Vault is a Flask-based web application that allows users to securely store and manage personal health records. Users can register, log in, upload medical reports, and view their health information anytime.

## Features
- User registration and login
- Upload medical records
- View health records
- User profile management
- Secure record storage

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite

## Requirements
Python 3.10+

## Installation
1. Clone the repository  
git clone https://github.com/KN-Swetha/phr-vault.git  

2. Navigate to the project folder  
cd phr-vault  

3. Install dependencies  
pip install -r requirements.txt  

4. Run the application  
python app.py  

5. Open in browser  
http://127.0.0.1:5000/
